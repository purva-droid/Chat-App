# Chat App

## Enhanced Features
- **Real-time Messaging:** Instant message delivery with WebSocket support.
- **User Authentication:** Secure login with OAuth.
- **Media Sharing:** Send images, videos, and documents.

## Quick Start
1. Clone the repository: `git clone https://github.com/purva-droid/Chat-App.git`
2. Navigate to the project directory: `cd Chat-App`
3. Install dependencies: `npm install`
4. Start the app: `npm start`

## Configuration
- **Port Settings:** Modify the `PORT` variable in `config.js` for your desired port.
- **Database Configuration:** Update `dbConfig` in `database.js` for your database connection.

## Troubleshooting
- Ensure Node.js and npm are installed.
- Check for port conflicts if the app doesn't start.

## Performance Tips
- Use a CDN for serving static files.
- Enable caching for static assets.
- Optimize images before uploading.
